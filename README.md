# 💻 Alan BADO | Portfolio de Développeur full stack

Ce dépôt contient le code source de mon portfolio professionnel, conçu pour présenter mes compétences et mes projets en tant que **Développeur Concepteur Logiciel (Titre RNCP Niveau 6)**.

## 🚀 Technologies Utilisées

Ce portfolio statique est construit avec les technologies fondamentales du développement web :

* **HTML5** : Structure sémantique du contenu.
* **CSS3** : Styles modernes et conception responsive.
* **Font Awesome** : Utilisation d'icônes (GitHub, LinkedIn, Enveloppe, etc.).

## ✨ Objectifs du Portfolio

1.  **Présenter mon Parcours** : Mettre en avant ma certification OpenClassrooms et mon objectif professionnel.
2.  **Mettre en Valeur mes Projets** : Fournir un accès direct aux dépôts des projets réalisés en Python (Django REST Framework, POO, etc.).
3.  **Faciliter le Contact** : Offrir des liens clairs vers mon profil LinkedIn et mes coordonnées.
